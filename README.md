## Description

DevDocs AI is a full-stack application designed to generate documentation for code snippets or specific functionalities in programming languages such as Java, PHP, Python, JavaScript, Go, Swift, and more.

The application allows users to generate reusable documentation that can be applied anywhere. By default, users receive a free subscription upon account creation, which comes with certain features. Users can upgrade to a paid subscription via Stripe to access additional capabilities.

- Free Subscription Features:

  - Create up to 2 projects.

  - Only public projects can be created.

  - Documentation can only be exported in plain text format.

  - Only the project owner can generate documentation.

- Premium Subscription Features::

  - Create up to 5 projects.

  - Create both public and private projects.

  - Add up to 2 users per project.

  - Members can rate the documentation generated by the project owner.

  - Documentation can be exported in plain text or PDF format.

Documentation is generated through an LLM provided by OpenRouter, which is currently free and allows generating comprehensive documentation as described above.

## Features

- App:

  - Regular user:

    - Authentication:

      - Login: Users enter their email and password to access the app.

      - Register: Users provide their details to create an account.

      - Activate Account: Users must enter their email and activation code to activate their account.

      - Resend Activation Code: Users can request a new activation code via email.

      - Reset Password: Users can reset their password by providing their email.

      - Assistance / via Admin: Users can request help for bugs, password resets, or activation code issues.

    - Dashboard:

      - View current subscription plan at any time.

      - Upgrade from Free to Premium.

      - Check the status of subscription payments:

        - If successful, view a button to download the invoice.

        - If failed, retry the payment process.

      - View the number of projects created and projects participated in.

    - Overview:

      - View all registered users in the app.

      - Invite new users if subscribed to Premium and if projects exist.

    - Projects:

      - Create new projects.

      - View all owned projects.

      - Filter projects by:

        - Privacy: Public or Private

        - Direct search: Text search field

      - View specified project:

        - Roles:

          - Owner:

            - Generate documentation within the daily LLM token limit.

            - Export documentation in plain text or PDF according to plan.

            - View, edit, or delete project information.

            - Add or remove users.

            - View and access generated documentation.

          - Member:

            - View project information.

            - Rate documentation.

            - View and access generated documentation.

    - LLM:

      - View LLM information accessible to all users.

      - Certain LLM restrictions and features are customized to enhance functionality in the app.

    - Notifications:

      - Two types of notifications:

        - Information: Project changes, user removal, or deletion of a project.

        - Action Required: Invitations to join projects (requires invitation code).

    - Settings:

      - Payment: View invoices for subscription upgrades via Stripe.

      - Profile: Access a "danger zone" to delete the user profile and all related data.

  - Admin User:

    - Authentication:

      - Login: Admins enter email and password to access the app.

      - Admin Panel:

        - Session Activity: View information about all user sessions.

        - Notifications:

          - Information: Notifications about bugs or app incidents.

          - Reset: Notifications about cache key resets when users exceed limits for password resets or activation code requests. Only the admin receives reset notifications.

## Technologies

- Backend

  - Python – Backend programming language

  - Django – Main backend framework (business logic, models, API, services)

  - FastAPI – Secondary backend framework (microservices / MITM)

  - PostgreSQL – Relational database

  - Celery – Asynchronous task queue

  - Redis – Cache management / broker for Celery

  - Jinja2 + MJML – Templates rendering (emails, dynamic content)

  - Flake8 – Python linter

- Frontend

  - Vue.js – Frontend framework

  - Pinia – State management

  - Vite – Development server

  - Tailwind – CSS framework

  - JavaScript / HTML / CSS – Core frontend languages

  - Prettier – Code formatter

  - ESLint – JavaScript linter

- Infrastructure

  - Docker – Containerization

  - Git Bash / Entrypoints – Scripts for container logic / startup

  - Stripe + Webhooks – Subscription payments

  - OpenRouter – LLM provider integration

  - GitHub Container Registry (GHCR) – Image registry for deployment

  - AWS Services:

    - IAM – Permissions and roles

    - S3 – Storage

    - CloudFront – CDN

    - EC2 – Compute instances

    - Systems Manager – Infrastructure management
